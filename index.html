¿QUÉ QUIERES SER DE MAYOR?

AUTORA: Ada García García

JUSTIFICACIÓN DE LA U.D.

​ Es importante que los alumnos conozcan las profesiones ​ y los oficios para conocer las actividades económicas que esto conlleva.

ETAPA/MATERIA

Esta Unidad Didáctica está planteada para alumnos/as de 1º de primaria. La materia será Ciencias Sociales. 

FORMACIÓN

En este caso se realizará de manera presencial ya que así podremos ayudar a aquel alumnado que tenga dudas. También porqué algunos ejercicios se podrán realizar conjuntamente, como el ejercicio de lectura (leerlo todos juntos).

De todos modos, esta U.D. está hecha de tal forma que se pueda realizar tanto presencial como online. Ya que también se puede hacer en casa perfectamente.
